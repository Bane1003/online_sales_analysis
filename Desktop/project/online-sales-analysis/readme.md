Overview
This is demonstration of GitHub & Python capbilities on real situations. We have here problems that need solving through branches, merging, conflicts.

Clasess:
PRODUCT
Product has attributes: name, price, quantity.
It's functionalities are product info and updating value.
PRODUCT MANAGER
It's attribut is a list of available products.
Functionalities are adding products, showing all products and value of all products.
MAIN
It has instance of product manager.
Functionalities are adding products, showing all of them and it"s values.
CART
Have a list for attributes.
Functionalities are adding in cart, counting sum of products and showing what is in cart.
With all this there are certain conflicts between them to resolve.
In the end there are gitignore database to know what not to show in repository.
